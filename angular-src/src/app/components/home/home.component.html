Income/expense logger

About

This is a simple income/expense logger made using MEAN stack.

Summary

-Application built using the MEAN-stack

-JSON Web Token based authentication

Features

-Registered users can log in, log out, create, read, update and delete incomes/expemses they own

-Incomes and expenses are displayed in a table and the user can filter and search their incomes/ expenses.

-Recent activity displayed in a line chart

Technologies used

-Node.js

-Express.js

-MongoDB with Mongoose

-Angular 7

-Chart.js for displaying data
